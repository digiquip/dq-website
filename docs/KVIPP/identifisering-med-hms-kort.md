# Identifisering med HMS-kort

I stedet for innlogging, bruker vi HMS-kort og annen ID (førerkort, pass) som identifisering. Med dette er det **ikke** nødvendig å legge til brukere eller opprette konto for å bruke Kvipp.

## HMS-kort og QR-kode
Når du skal bruke HMS-kort: Bruk knappen QR-kode og skann QR-koden på HMS-kortet. Systemet finner automatisk navn og arbeidsgiver, og slik blir opplæringen automatisk koblet til både riktig person og virksomhet. Systemet sørger for at alt blir dokumentert og systematisert.

## HMS API tilgang
Med tilgang til Arbeidstilsynets API henter vi fornavn, etternavn, fødselsdato og arbeidsgiver. 

"QR-koden på kortet viser den statusen som er registrert i de offentlige registrene idet koden blir skannet." Les mer om dette her: https://www.arbeidstilsynet.no/hms/hms-kort/ 

## Andre identitetskort 
Har dere ikke HMS-kort? Alternativ til HMS-kort er førerkort, pass eller eventuelt Kvipp ID. 

Når du skal bruke førerkort og pass: Bruk knappen bilde og ta bilde av kortet. Du må få med fornavn, etternavn og fødselsdato. 

## Er det problematisk at hvem som helst kan skanne HMS-kortet?
Nei. 

Det er ikke slik at hvem som helst skal ha mulighet til å skanne HMS-kortet. Det er tydelige retningslinjer om hvem som kan skanne HMS-kort. Dette er beskrevet på Arbeidstilsynets hjemmesider, https://www.arbeidstilsynet.no/hms/hms-kort/. 

Det er arbeidstakers ansvar å ta vare på HMS-kortet. Blant annet stiller Arbeidstilsynet krav om at det umiddelbart skal meldes tapt og at arbeidstaker kan bli straffet om HMS-kortet lånes bort. 

Med disse retningslinjene fra Arbeidstilsynet, er det tydelig at HMS-kortet ikke må misbrukes. Som også gjelder identifisering av personer i forbindelse med opplæring.

## Er det problematisk at vi bare kan skanne et ID-kort?
Nei. 

Alle som har HMS-kort, førerkort og pass har ansvar for å ta vare på disse kortene og hvem som får tilgang. Som med for eksempel førerkort, er det mulig å sjekke opp andres kompetanse via Kvipp.

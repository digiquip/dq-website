Updates 2025

DigiQuip-Inspection Date: 18.03.2025
Unlock Function
Administrators and the original inspector can now unlock completed inspections.
Save and Close Function
Inspections that are not completed are automatically saved as drafts.
Drafts have the status "Not Inspected" and will not appear on the QR code landing page until finalized.
Pre-filling "OK" in Checklists
A new toggle function allows users to enable or disable the pre-filling of "OK" in checklists.
This provides better control if an inspection is not completed immediately but needs to be continued later. More info can be found here.
Retrieving Documents from BraReg
All documents related to the equipment and production year are now fetched directly from BraReg.
Bug Fixes
Corrected status in DigiQuip when equipment is set to "Not Approved" due to a Level 2 defect.
Fixed an issue preventing some equipment from properly linking to BraReg.
Resolved a synchronization issue for inspections from BraReg that already had an existing connection.
